# Vocabulab Engineering Rules
- Architecture: Clean Architecture / SOLID.
- Framework: FastAPI (Python 3.11+).
- Deployment: Docker Multi-stage builds.
- Storage: SQLite (persistence via Docker volumes).
- Language: French (NLP via spaCy fr_core_news_sm), leave space to add languages in the future.
- Environment: ZimaOS (Root-only partition is read-only).
- Principle: No "toy code". Use error handling, logging, and performance best practices.
